# Technical readme

1. Set working directory
1. library(devtools)` - Load devtools package
1. create("package_name") - Create an R project that contains main components for an R package
1. Set up information in the DESCRIPTION file
1. use_r("r_file_name") - Create an R script and place it into R sub-folder
1. Open r_file_name.R script and add desired functions
1. load_all() - load all functions in the package

Guide from https://rcs.bu.edu/examples/r/tutorials/BuildingPackages/r-project-setup.html#summary-create-new-r-package-workflow
